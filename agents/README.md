# Agents

**Purpose:** Agent prompts and role definitions for structured contributions.

**Expected file types:** Markdown agent briefs.

**Status:** Active.

## Available Agents

- `brainstorm_agent.md` - ideation only; does not write prose.
- `outline_agent.md` - narrative architecture only; does not write prose.
- `revision_agent.md` - critique and revision planning; rewrites only when explicitly instructed.
- `short_story_drafting_agent.md` - short-story prose drafting plus repo hygiene for entities, lexicon, timeline, generated artifacts, and validation.
